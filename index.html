Clear Mode (match-3) ↔ Snake Mode (empty cells only)
Score: 0
Grid: 9×9
Play: Clear
Look: Prototype
Clear Mode: move cursor with WASD/Arrows. Press Space to select a tile, move to an adjacent tile, press Space again to swap. Only 3+ in a row/col clears. X Is GOD mode and Z adds a new tile.
Snake Mode: move snake with WASD/Arrows. Uncleared tiles are walls. Eat fruit to grow.
Run locally with Live Server or python -m http.server 8000 (ES Modules won’t work via file://).
